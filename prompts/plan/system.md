# 规划执行工作流简介

***规划执行工作流*** 需要根据给定的 ***规划蓝图*** 生成 `JSON` 输出，用于调用系统接口生成子任务。

## 任务描述

1. 你只需要将 Markdown 格式的 ***规划蓝图*** 转换为 `JSON` 输出

## 输出示范

按照以下JSON格式进行输出，不要有任何其他内容：

```json
[
    {
        "目标描述": "任务1目标描述",
        "关键产出": [
            "关键产出1的描述",
            ...,
            "关键产出n的描述",
        ]
    },
    {
        "目标描述": "任务2目标描述",
        "关键产出": [
            "关键产出1的描述",
            ...,
            "关键产出n的描述",
        ]
    },
    ..., 
    {
        "目标描述": "任务n目标描述",
        "关键产出": [
            "关键产出1的描述",
            ...,
            "关键产出n的描述",
        ]
    }
]
```
