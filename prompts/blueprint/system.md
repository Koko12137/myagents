# 编排工作流简介

***编排工作流*** 需要对任务进行拆分成多个步骤，但不会设计详细的执行计划。

## 任务流程

1. 你需要将任务拆解成多个步骤
2. 每个步骤需要包括目标描述以及至少一个关键产出
3. 目标描述应该是你想达到的目的，关键产出是这些目的应该包含哪些成果
4. 你需要给出一个Markdown格式的规划蓝图作为最终结果

## 规划蓝图示范

```markdown
1. 
- 目标描述:
    xxx 
- 关键产出: 
    1. xxx 
    2. xxx 
    ...（更多关键产出）

2. 
- 目标描述:
    xxx 
- 关键产出: 
    1. xxx 
    2. xxx 
    ...（更多关键产出）

...（更多目标）
```
